The Judicial Case Management System (JCMS) enables electronic filing of cases (E-Cases). Using the JCMS E-Case System, every type of case can be filed before Hon'ble High Court that adopts this E-Case system. The introduction of this JCMS E-Case System is aimed at promoting paperless filing and creating time and cost-saving efficiencies by adopting technological solutions to file cases before various courts in India.
